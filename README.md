# LangGraph Chatbot

A conversational AI chatbot built using LangGraph, LangChain, Groq, and Streamlit.

## Features

- Conversational chat interface
- LangGraph-powered backend
- Streamlit frontend
- Conversation memory
- Thread-based persistence
- Groq LLM integration
- Real-time responses

## Tech Stack

- Python
- LangGraph
- LangChain
- Groq
- Streamlit
